TimeMachin does not allow to delete individual files from another machine’s backup using regular commands. However, old backups can be deleted using “tmutil” command

e.g

sudo tmutil "/Volumes/Time Machine Backups/Backups.backupdb/computer_name/backup_date"

This will delete all files in the backup for that particular backup. 

However, to delete files inside of a backup is not possible using this method since backups are protected at kernel level. To bypass this protected, apple provides a utility called “bypass” :) - any command passed to this utility will bypass the protection.

on Yosemite

sudo /System/Library/Extensions/TMSafetyNet.kext/Contents/Helpers/bypass rm -rf "/Volumes/Time Machine Backups/Backups.backupdb/computer_name/backup_date/Macintosh HD/Users/me/Desktop/somefile.jpg"

on Mountain Lion, it exists in

/System/Library/Extensions/TMSafetyNet.kext/Helpers/bypass

on Lion,

/System/Library/Extensions/TMSafetyNet.kext/Contents/MacOS/bypass
